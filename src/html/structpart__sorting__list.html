Detailed Description

User defined data object that holds box metadata and the number of parts that remain to be sorted.

This struct acts as a single point of data manipulation for the algo. It contains the list of all the box_metadata strct or each type of box with their initial values. part_sorting_list.currentbox is a pointer that always points to the box that parts need to be sorted into. If the pointer ends up being a NULL then we have reached the end of the list
